My Interests in Accessibility

I believe that accessibility should not be an afterthought in development. Accessibility should be a component of proper code design. Some things I am interested in are:

  • Keyboard navigation
  • Color contrast and readability
  • Screen reader compatibility

My accessibility focus

Accessibility has not always been something that I thought much of. However, I have been trying to get better at keyboard only navigation in an effort to use my mouse less. I was surprised how many sites supported keyboard navigation and how useful some of these elements are. For example, "Skip to Main Content" links and headers are so useful for quick navigation. Realizing that these exist due to accessibility made me realize how much accessibility makes everyone's lives easier.
